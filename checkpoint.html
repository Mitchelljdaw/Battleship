Game Logic

Incrementing the turns everytime a cell in click on:Done
Click on a cell and gameState.turns it incremented by one everytime a cell is clicked on.
Changing the the turn each time a cell is click on:Done
Click on a cell and it will change the gameState.turn to either 'yours' or 'opp' based on the amount of turns that have been taken.
Check to see if all ships have been placed before allowing game to start:Done
Click the sart button and the gameStart function from the model is called. If all ships have been placed the game will start.If all the ships have not been placed a message will be displayed in the debug div asking user to place all ships.
Check to see if the game is over:Done
Click on a cell and the gameOver function from the model will be called. If all of either your ships or the oppnants damage properties equal their size properties then an alert will let the user know who won.
Hits and misses are marked on each grid:Done
Click on a cell and it will check if a ship has been placed there and if it has it will turn the cell back. If it has not the cell will turn red.
